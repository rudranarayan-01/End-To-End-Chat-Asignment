# Sync - A Chat Application
A professional, real-time messaging platform built with a modern full-stack architecture. Nexus allows users to register, discover other users, and engage in secure, isolated, one-to-one conversations with instant message delivery.

## Features
    - Real-Time Messaging: Powered by Flask-SocketIO and Socket.io-client for instant, low-latency communication.

    - Professional UI/UX: Built with React 18, Vite, Tailwind CSS, and Shadcn UI for a high-end, responsive aesthetic.

    - Authentication: Secure Login and Signup system with password hashing via Bcrypt.

    - Persistent Storage: All users and message histories are stored in a SQLite database using SQLAlchemy.

    - Isolated Conversations: Chat history is private and unique between specific user pairs.

    - Dynamic UI: Auto-scrolling chat windows, glassmorphism headers, and "Online" status indicators.
    

## Tech Stack
### Frontend:

    React (TypeScript)

    Vite

    Tailwind CSS

    Shadcn UI

    Lucide React (Icons)

    Axios (API Calls)

### Backend:

    Python Flask

    Flask-SocketIO (WebSockets)

    Flask-SQLAlchemy (ORM)

    Flask-Bcrypt (Security)

    Flask-CORS
